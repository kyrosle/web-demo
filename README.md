## 安装

0. git clone https://github.com/swiftdo/web-demo
1. 安装 fvm 工具: [leoafarias/fvm](https://github.com/leoafarias/fvm)
2. 运行 

   ```sh
   $ cd web-demo
   $ fvm install
   $ fvm flutter run -d Chrome
   ```
